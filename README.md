# Excel_jobs_salary_analysis
Excel Dashboard and Data analysis project using table, formulas, stats, Charts, KPI.
This project analysis Data jobs trend exploring jobs based on salary, median salary, country, job type, also mentions KPIs like Top Platform, Job count, median salary.
Skills used- Tables, Data cleaning, charts, KPI Dashboard, Basis data analysis, Statistics.
Insights- Highest Paying jobs by country, data jobs count, Top job platform, Median salary.
